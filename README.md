# Blockchain Transaction System

**Description**  
A full-stack **blockchain project** built solo, simulating secure money transactions and block creation. Developed with React, Redux Toolkit, HTML, CSS, and JavaScript for the frontend, Node.js + Express for the backend, and PostgreSQL for storing transactions and blocks.

This system allows users to:
✔️ Create and validate transactions  
✔️ Generate new blocks and append them to the blockchain  
✔️ View transaction history and block data  
✔️ Interact with a responsive frontend interface  

## 🧑‍💻 My Role — Solo Developer
As the sole developer, I:  
- Designed and implemented the **block creation logic**, including hashing and chain linking  
- Built the **transaction management system** between wallets  
- Developed RESTful APIs in **Express.js** for blockchain data  
- Built the **React + Redux Toolkit** frontend for real-time blockchain visualization  
- Designed and managed the **PostgreSQL** database schema  
- Ensured secure and stable communication between UI and server  

## 🚀 Tech Stack
- **Frontend:** React, Redux Toolkit, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL  
- **Blockchain Concepts:** Block hashing, chain linking, transaction validation  

## 📦 Features
- Add new transactions  
- Create new blocks  
- Maintain an immutable chain of blocks  
- Dynamic UI for transaction and block visualization
